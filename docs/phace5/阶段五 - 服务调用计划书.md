## 📚 核心概念

### 1. 什么是 OpenFeign？

```
┌─────────────────────────────────────────────────────────────┐
│                    User 服务 (9001)                          │
│                                                              │
│   // 传统方式：RestTemplate                                  │
│   restTemplate.getForObject("http://course:9002/...", ...); │
│                                                              │
│   // OpenFeign 方式：声明式接口                             │
│   @FeignClient("yunh-service-course")                        │
│   interface CourseClient {                                   │
│       @GetMapping("/course/{id}") Course getCourseById(...);│
│   }                                                          │
│   // 调用：courseClient.getCourseById(1L); ← 像本地方法调用  │
└─────────────────────────────────────────────────────────────┘
```

**OpenFeign 的优势：**

| 对比项 | RestTemplate | OpenFeign |
|--------|-------------|-----------|
| **代码风格** | 命令式，模板化 | 声明式，接口化 |
| **可读性** | 较低，URL 硬编码 | 高，接口定义清晰 |
| **维护性** | URL 变更需多处修改 | 只需改接口定义 |
| **集成度** | 需手动集成负载均衡 | 自动集成 Ribbon/LoadBalancer |
| **可测试性** | 较难 Mock | 容易 Mock 测试 |

### 2. OpenFeign 工作原理

```
┌─────────────────────────────────────────────────────────────────┐
│  调用：courseClient.getCourseById(1L)                            │
└─────────────────────────────────────────────────────────────────┘
                              │
                              ▼
┌─────────────────────────────────────────────────────────────────┐
│  Feign 动态代理                                                  │
│  1. 解析 @FeignClient 注解                                       │
│  2. 解析 @GetMapping 等映射注解                                   │
│  3. 构建 HTTP 请求                                                │
└─────────────────────────────────────────────────────────────────┘
                              │
                              ▼
┌─────────────────────────────────────────────────────────────────┐
│  Spring Cloud LoadBalancer                                       │
│  从 Nacos 获取服务列表，选择实例                                  │
└─────────────────────────────────────────────────────────────────┘
                              │
                              ▼
┌─────────────────────────────────────────────────────────────────┐
│  Course 服务 (9002)                                              │
│  返回 Course 对象                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### 步骤 5.2：创建 Feign Client 接口（预计 20 分钟）

#### 5.2.1 在 yunh-api 模块中定义 Feign Client（推荐方式）

**为什么放在 api 模块？**
- ✅ 避免循环依赖
- ✅ 服务间共享接口定义
- ✅ 便于版本管理

```java
// yunh-api/src/main/java/com/yunh/api/client/CourseClient.java
package com.yunh.api.client;

import com.yunh.pojo.vo.CourseVO;
import org.springframework.cloud.openfeign.FeignClient;
import org.springframework.web.bind.annotation.GetMapping;
import org.springframework.web.bind.annotation.PathVariable;

import java.util.List;

/**
 * 课程服务 Feign Client
 * 注意：FeignClient 的 name 必须与 Nacos 中的服务名一致
 */
@FeignClient(name = "yunh-service-course")
public interface CourseClient {
    
    /**
     * 根据 ID 获取课程详情
     */
    @GetMapping("/course/{id}")
    CourseVO getCourseById(@PathVariable("id") Long id);
    
    /**
     * 根据讲师 ID 获取课程列表
     */
    @GetMapping("/course/teacher/{teacherId}")
    List<CourseVO> getCoursesByTeacherId(@PathVariable("teacherId") Long teacherId);
    
    /**
     * 批量获取课程信息
     */
    @GetMapping("/course/batch")
    List<CourseVO> getCoursesByIds(@PathVariable("ids") List<Long> ids);
}
```

### 步骤 5.3：实现服务调用业务逻辑（预计 25 分钟）

#### 5.3.1 User 服务调用 Course 服务

```java
// yunh-service-user/src/main/java/.../service/UserCourseService.java
package com.yunh.user.service;

import com.yunh.api.client.CourseClient;
import com.yunh.pojo.vo.CourseVO;
import com.yunh.pojo.vo.UserCourseVO;
import lombok.RequiredArgsConstructor;
import lombok.extern.slf4j.Slf4j;
import org.springframework.stereotype.Service;

import java.util.List;
import java.util.stream.Collectors;

@Slf4j
@Service
@RequiredArgsConstructor
public class UserCourseService {
    
    private final CourseClient courseClient;
    
    /**
     * 获取用户选的课程列表（带课程详情）
     */
    public List<UserCourseVO> getUserCoursesWithDetails(Long userId) {
        // 1. 查询用户选课关系（本地数据库）
        List<Long> courseIds = findCourseIdsByUserId(userId);
        
        if (courseIds.isEmpty()) {
            return List.of();
        }
        
        // 2. 通过 Feign 调用 Course 服务获取课程详情
        List<CourseVO> courses = courseClient.getCoursesByIds(courseIds);
        
        // 3. 组装返回结果
        return courses.stream()
            .map(course -> UserCourseVO.builder()
                .courseId(course.getId())
                .courseName(course.getName())
                .courseCover(course.getCover())
                .teacherName(course.getTeacherName())
                .build())
            .collect(Collectors.toList());
    }
    
    private List<Long> findCourseIdsByUserId(Long userId) {
        // TODO: 从数据库查询用户选课关系
        return List.of(1L, 2L, 3L); // 示例数据
    }
}
```

// yunh-service-course/src/main/java/.../service/CourseUserService.java
package com.yunh.course.service;

import com.yunh.api.client.UserClient;
import com.yunh.pojo.vo.UserVO;
import com.yunh.pojo.vo.CourseDetailVO;
import lombok.RequiredArgsConstructor;
import lombok.extern.slf4j.Slf4j;
import org.springframework.stereotype.Service;

@Slf4j
@Service
@RequiredArgsConstructor
public class CourseUserService {
    
    private final UserClient userClient;
    
    /**
     * 获取课程详情（带讲师信息）
     */
    public CourseDetailVO getCourseDetailWithTeacher(Long courseId) {
        // 1. 查询课程基本信息（本地数据库）
        CourseDetailVO course = findCourseById(courseId);
        
        // 2. 通过 Feign 调用 User 服务获取讲师信息
        Long teacherId = course.getTeacherId();
        if (teacherId != null) {
            try {
                UserVO teacher = userClient.getUserById(teacherId);
                course.setTeacherName(teacher.getNickname());
                course.setTeacherAvatar(teacher.getAvatar());
                course.setTeacherIntro(teacher.getIntro());
            } catch (Exception e) {
                log.warn("获取讲师信息失败，courseId: {}, error: {}", courseId, e.getMessage());
                // 降级处理：设置默认值
                course.setTeacherName("未知讲师");
            }
        }
        
        return course;
    }
    
    private CourseDetailVO findCourseById(Long courseId) {
        // TODO: 从数据库查询课程信息
        return CourseDetailVO.builder()
            .id(courseId)
            .name("Spring Cloud 微服务实战")
            .teacherId(1L)
            .build();
    }
}

#### 5.4.1 在 Nacos 配置中心配置 Feign 参数

```yaml
# OpenFeign 配置
feign:
  # 客户端配置
  client:
    config:
      default:  # 全局配置
        # 连接超时时间（毫秒）
        connectTimeout: 5000
        # 读取超时时间（毫秒）
        readTimeout: 10000
        # 日志级别
        loggerLevel: BASIC
        # 重试配置（可选）
        retryer:
          period: 1000
          maxPeriod: 2000
          maxAttempts: 3
      
      # 针对特定服务的配置
      yunh-service-course:
        readTimeout: 15000  # 课程服务读取超时更长
      yunh-service-user:
        readTimeout: 8000   # 用户服务读取超时较短

  # HTTP 客户端配置（使用 OkHttp）
  okhttp:
    enabled: true

# OkHttp 连接池配置
okhttp:
  connect-timeout: 5000
  read-timeout: 10000
  write-timeout: 10000
  max-idle-connections: 50
  keep-alive-duration: 300000

# 服务调用超时（Spring Cloud LoadBalancer）
ribbon:
  ReadTimeout: 10000
  ConnectTimeout: 5000
```

#### 5.4.2 Feign 日志级别说明

| 级别 | 说明 | 适用场景 |
|------|------|---------|
| `NONE` | 不记录日志 | 生产环境（高性能） |
| `BASIC` | 记录请求方法、URL、响应状态、执行时间 | 生产环境（推荐） |
| `HEADERS` | BASIC + 请求头和响应头 | 测试环境 |
| `FULL` | HEADERS + 请求体和响应体 | 开发调试 |

package com.yunh.user.client.fallback;

import com.yunh.api.client.CourseClient;
import com.yunh.pojo.vo.CourseVO;
import lombok.extern.slf4j.Slf4j;
import org.springframework.stereotype.Component;

import java.util.Collections;
import java.util.List;

/**
 * CourseClient 的降级实现
 * 当 Course 服务不可用时，提供兜底数据
 */
@Slf4j
@Component
public class CourseFallback implements CourseClient {
    
    @Override
    public CourseVO getCourseById(Long id) {
        log.error("获取课程详情失败，courseId: {}", id);
        // 返回兜底数据
        return CourseVO.builder()
            .id(id)
            .name("课程暂不可用")
            .build();
    }
    
    @Override
    public List<CourseVO> getCoursesByTeacherId(Long teacherId) {
        log.error("获取讲师课程列表失败，teacherId: {}", teacherId);
        return Collections.emptyList();
    }
    
    @Override
    public List<CourseVO> getCoursesByIds(List<Long> ids) {
        log.error("批量获取课程信息失败，ids: {}", ids);
        return Collections.emptyList();
    }
}

### 步骤 5.6：配置 Feign 拦截器（预计 15 分钟）

#### 5.6.1 传递用户上下文（如 Token）

```java
// yunh-service-user/src/main/java/.../config/FeignConfig.java
package com.yunh.user.config;

import feign.RequestInterceptor;
import org.springframework.context.annotation.Bean;
import org.springframework.context.annotation.Configuration;
import org.springframework.web.context.request.RequestContextHolder;
import org.springframework.web.context.request.ServletRequestAttributes;

import jakarta.servlet.http.HttpServletRequest;

/**
 * Feign 配置类
 * 注意：不要在这个类上加 @Configuration，会被全局扫描
 * 应该在 @FeignClient 的 configuration 属性中指定
 */
@Configuration
public class FeignConfig {
    
    /**
     * 请求拦截器：传递 Authorization Header
     */
    @Bean
    public RequestInterceptor authorizationRequestInterceptor() {
        return template -> {
            ServletRequestAttributes attributes = 
                (ServletRequestAttributes) RequestContextHolder.getRequestAttributes();
            if (attributes != null) {
                HttpServletRequest request = attributes.getRequest();
                String token = request.getHeader("Authorization");
                if (token != null) {
                    template.header("Authorization", token);
                }
            }
        };
    }
    
    /**
     * 传递自定义 Header（如请求 ID 用于链路追踪）
     */
    @Bean
    public RequestInterceptor requestIdRequestInterceptor() {
        return template -> {
            ServletRequestAttributes attributes = 
                (ServletRequestAttributes) RequestContextHolder.getRequestAttributes();
            if (attributes != null) {
                HttpServletRequest request = attributes.getRequest();
                String requestId = request.getHeader("X-Request-ID");
                if (requestId != null) {
                    template.header("X-Request-ID", requestId);
                }
            }
        };
    }
}
```

## ⚠️ 注意事项

### 1. 循环依赖问题
- ❌ 错误：UserClient 在 user 模块，CourseClient 在 course 模块
- ✅ 正确：所有 FeignClient 接口放在独立的 api 模块

### 2. FeignClient 的 name 属性
必须与 Nacos 中的服务名**完全一致**：
```java
@FeignClient(name = "yunh-service-course")  // 不是 "course-service"
```

### 3. 路径匹配
Feign 的 `@GetMapping` 路径要与 Controller 的路径**完全匹配**

### 4. 参数传递
- 路径参数：`@PathVariable`（必须指定 name）
- 查询参数：`@RequestParam`
- 请求体：`@RequestBody`（一个接口只能有一个）

### 5. 超时配置
生产环境必须配置超时，避免线程池耗尽：
```yaml
feign:
  client:
    config:
      default:
        connectTimeout: 5000
        readTimeout: 10000
```
