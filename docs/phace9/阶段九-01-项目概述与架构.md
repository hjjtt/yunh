# 阶段九-01：项目概述与架构

## 🎯 项目目标

**打造一个真正可用的企业级在线教育系统**，综合运用阶段一至阶段八的所有技术，并引入生产级最佳实践。

***

## 📚 项目概述

### 项目名称

**云课堂 2.0 (YunH 2.0)** - 企业级在线教育平台

### 项目定位

- **To C**：学生在线学习、课程购买、视频播放
- **To B**：讲师课程管理、学员管理、数据分析
- **To Admin**：平台运营、内容审核、数据统计

## 🏗️ 系统架构

### 技术栈总览

| 层级      | 技术选型                    | 说明       |
| ------- | ----------------------- | -------- |
| **前端**  | Vue3 + Element Plus     | 管理后台     |
| **移动端** | UniApp                  | 小程序/H5   |
| **网关层** | Spring Cloud Gateway    | 路由、鉴权、限流 |
| **服务层** | Spring Cloud Alibaba    | 微服务核心    |
| **数据层** | MySQL + Redis           | 主从 + 缓存  |
| **搜索**  | Elasticsearch           | 课程搜索     |
| **消息**  | RocketMQ                | 异步解耦     |
| **监控**  | SkyWalking + Prometheus | 全链路监控    |
| **部署**  | Docker + Kubernetes     | 容器化部署    |

## 📁 微服务拆分

### 服务划分

```
yunh-cloud-2.0/
├── yunh-gateway/              # 网关服务 (9000)
├── yunh-auth/                 # 认证服务 (9001)
├── yunh-user/                 # 用户服务 (9002)
├── yunh-course/               # 课程服务 (9003)
├── yunh-order/                # 订单服务 (9004)
├── yunh-pay/                  # 支付服务 (9005)
├── yunh-video/                # 视频服务 (9006)
├── yunh-interaction/          # 互动服务 (9007)
├── yunh-search/               # 搜索服务 (9008)
├── yunh-statistics/           # 统计服务 (9009)
└── yunh-common/               # 公共模块
    ├── yunh-common-core/      # 核心工具
    ├── yunh-common-redis/     # Redis 工具
    ├── yunh-common-security/  # 安全工具
    └── yunh-common-swagger/   # API 文档
```

### 服务职责

| 服务              | 端口   | 职责            | 数据库               |
| --------------- | ---- | ------------- | ----------------- |
| **Gateway**     | 9000 | 网关路由、鉴权、限流    | -                 |
| **Auth**        | 9001 | 登录认证、Token 管理 | yunh\_auth        |
| **User**        | 9002 | 用户信息、个人中心     | yunh\_user        |
| **Course**      | 9003 | 课程管理、章节视频     | yunh\_course      |
| **Order**       | 9004 | 订单管理、购物车      | yunh\_order       |
| **Pay**         | 9005 | 支付对接、退款       | yunh\_pay         |
| **Video**       | 9006 | 视频上传、转码、播放    | yunh\_video       |
| **Interaction** | 9007 | 评论、问答、笔记      | yunh\_interaction |
| **Search**      | 9008 | 课程搜索、推荐       | ES                |
| **Statistics**  | 9009 | 数据统计、报表       | yunh\_statistics  |
