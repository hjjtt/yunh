## 步骤 9.2：认证服务

### 9.2.1 技术选型

| 方案 | 说明 | 选择 |
|------|------|------|
| Spring Security | Spring 官方安全框架 | ❌ 较重 |
| Apache Shiro | 轻量级安全框架 | ❌ 功能有限 |
| **Sa-Token** | 国产轻量级权限框架 | ✅ 推荐 |

### 9.2.2 核心功能

```java
// 登录接口
POST /api/auth/login
{
    "username": "user@example.com",
    "password": "123456",
    "captcha": "abcd"
}

// 返回
{
    "code": 200,
    "data": {
        "token": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...",
        "expiresIn": 7200,
        "refreshToken": "dGhpcyBpcyBhIHJlZnJlc2ggdG9rZW4..."
    }
}

// 刷新 Token
POST /api/auth/refresh
{
    "refreshToken": "dGhpcyBpcyBhIHJlZnJlc2ggdG9rZW4..."
}

// 登出
POST /api/auth/logout
```

### 9.2.3 登录流程

```
┌─────────────────────────────────────────────────────────────┐
│                      登录认证流程                            │
│                                                              │
│  用户 ──→ 输入账号密码 ──→ 验证码                           │
│                              │                               │
│                              ▼                               │
│  ┌─────────────────────────────────────────────────────┐    │
│  │                  Auth 服务                           │    │
│  │  1. 验证验证码                                       │    │
│  │  2. 验证账号密码                                      │    │
│  │  3. 生成 Token + RefreshToken                         │    │
│  │  4. 记录登录日志                                      │    │
│  │  5. 返回 Token                                        │    │
│  └─────────────────────────────────────────────────────┘    │
│                              │                               │
│                              ▼                               │
│  用户保存 Token ──→ 后续请求携带 Token ──→ Gateway 鉴权      │
│                                                              │
└─────────────────────────────────────────────────────────────┘
```

## 步骤 9.3：网关统一鉴权

### 9.3.1 鉴权过滤器

```java
@Component
public class AuthGlobalFilter implements GlobalFilter, Ordered {

    @Override
    public Mono<Void> filter(ServerWebExchange exchange, GatewayFilterChain chain) {
        ServerHttpRequest request = exchange.getRequest();
        String path = request.getPath().value();

        // 1. 白名单放行
        if (isWhiteList(path)) {
            return chain.filter(exchange);
        }

        // 2. 获取 Token
        String token = request.getHeaders().getFirst("Authorization");

        // 3. 调用 Auth 服务验证 Token
        // 通过 Feign 调用：authClient.validateToken(token)

        // 4. 验证通过，传递用户信息到下游
        ServerHttpRequest mutatedRequest = request.mutate()
            .header("X-User-Id", userId)
            .header("X-Username", username)
            .build();

        return chain.filter(exchange.mutate().request(mutatedRequest).build());
    }
}
```

### 9.3.2 限流配置

```yaml
# Nacos 配置：yunh-gateway.yml
spring:
  cloud:
    gateway:
      routes:
        - id: yunh-auth
          uri: lb://yunh-auth
          predicates:
            - Path=/api/auth/**
          filters:
            - name: RequestRateLimiter
              args:
                redis-rate-limiter.replenishRate: 10  # 每秒补充 10 个令牌
                redis-rate-limiter.burstCapacity: 20  # 桶容量 20
                key-resolver: "#{@ipKeyResolver}"
```

## 步骤 9.4：课程服务

### 9.4.1 课程表设计

```sql
CREATE TABLE course (
    id BIGINT PRIMARY KEY AUTO_INCREMENT,
    title VARCHAR(200) NOT NULL COMMENT '课程标题',
    subtitle VARCHAR(500) COMMENT '课程副标题',
    cover VARCHAR(500) COMMENT '课程封面',
    price DECIMAL(10,2) DEFAULT 0 COMMENT '价格',
    original_price DECIMAL(10,2) COMMENT '原价',
    teacher_id BIGINT NOT NULL COMMENT '讲师 ID',
    teacher_name VARCHAR(100) COMMENT '讲师姓名',
    category_id BIGINT COMMENT '分类 ID',
    status TINYINT DEFAULT 0 COMMENT '状态：0-草稿 1-上架 2-下架',
    sales INT DEFAULT 0 COMMENT '销量',
    study_count INT DEFAULT 0 COMMENT '学习人数',
    rating DECIMAL(2,1) DEFAULT 0 COMMENT '评分',
    create_time DATETIME DEFAULT CURRENT_TIMESTAMP,
    update_time DATETIME DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP,
    INDEX idx_category (category_id),
    INDEX idx_teacher (teacher_id),
    INDEX idx_status (status)
);

CREATE TABLE course_chapter (
    id BIGINT PRIMARY KEY AUTO_INCREMENT,
    course_id BIGINT NOT NULL COMMENT '课程 ID',
    title VARCHAR(200) NOT NULL COMMENT '章节标题',
    sort INT DEFAULT 0 COMMENT '排序',
    create_time DATETIME DEFAULT CURRENT_TIMESTAMP
);

CREATE TABLE course_video (
    id BIGINT PRIMARY KEY AUTO_INCREMENT,
    chapter_id BIGINT NOT NULL COMMENT '章节 ID',
    course_id BIGINT NOT NULL COMMENT '课程 ID',
    title VARCHAR(200) NOT NULL COMMENT '视频标题',
    video_url VARCHAR(500) COMMENT '视频地址',
    duration INT COMMENT '时长（秒）',
    is_free TINYINT DEFAULT 0 COMMENT '是否免费',
    sort INT DEFAULT 0 COMMENT '排序',
    create_time DATETIME DEFAULT CURRENT_TIMESTAMP
);
```
