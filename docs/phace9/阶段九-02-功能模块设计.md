# 阶段九-02：功能模块设计

---

## 模块一：用户系统（User Service）

```
┌─────────────────────────────────────────────────────────────┐
│                      用户系统功能                            │
│                                                              │
│  ┌─────────────────┐  ┌─────────────────┐                   │
│  │   注册登录      │  │   个人中心      │                   │
│  │  - 手机号注册   │  │  - 基本信息     │                   │
│  │  - 密码登录     │  │  - 头像上传     │                   │
│  │  - 短信验证码   │  │  - 修改密码     │                   │
│  │  - 微信登录     │  │  - 学习记录     │                   │
│  └─────────────────┘  └─────────────────┘                   │
│                                                              │
│  ┌─────────────────┐  ┌─────────────────┐                   │
│  │   学习记录      │  │   我的收藏      │                   │
│  │  - 已学课程     │  │  - 收藏课程     │                   │
│  │  - 学习进度     │  │  - 收藏讲师     │                   │
│  │  - 学习时长     │  │  - 收藏笔记     │                   │
│  │  - 学习证书     │  │  - 收藏问答     │                   │
│  └─────────────────┘  └─────────────────┘                   │
└─────────────────────────────────────────────────────────────┘
```

**核心接口：**
```java
POST   /api/user/register          // 用户注册
POST   /api/user/login             // 用户登录
GET    /api/user/profile           // 获取个人信息
PUT    /api/user/profile           // 更新个人信息
POST   /api/user/avatar            // 上传头像
GET    /api/user/learning-record   // 学习记录
GET    /api/user/favorites         // 收藏列表
```

│  │  - 课程分类     │  │  - 课程大纲     │                   │
│  │  - 课程标签     │  │  - 讲师信息     │                   │
│  │  - 课程搜索     │  │  - 学员评价     │                   │
│  └─────────────────┘  └─────────────────┘                   │
│                                                              │
│  ┌─────────────────┐  ┌─────────────────┐                   │
│  │   讲师管理      │  │   课程管理      │                   │
│  │  - 讲师入驻     │  │  - 创建课程     │                   │
│  │  - 讲师认证     │  │  - 章节管理     │                   │

**核心接口：**
```java
GET    /api/course/list            // 课程列表
GET    /api/course/{id}            // 课程详情
GET    /api/course/{id}/chapters   // 课程章节
POST   /api/course                 // 创建课程（讲师）
PUT    /api/course/{id}            // 更新课程（讲师）
POST   /api/course/{id}/publish    // 发布课程（讲师）
GET    /api/teacher/{id}/courses   // 讲师课程列表
```
