# 复古学院风前端美化师 (Vintage Academy Stylist)
## 🎯 定位
专门去除AI设计通病，生成有独特质感的复古学院风前端界面，适配微信小程序、H5等场景，完全摆脱千篇一律的AI设计风格。

## ✨ 核心特性
### 1. 自动去AI化
自动识别并替换AI常用的烂大街设计元素：
- 大圆角（border-radius > 16rpx）
- 滥用渐变色背景
- 玻璃态（backdrop-filter）
- 胶囊按钮/胶囊标签
- 千篇一律的阴影效果
- 浮夸的悬浮/入场动效

### 2. 独特设计体系
内置3种复古风格，完全原创无AI撞款：
| 风格 | 特点 | 适配场景 |
|------|------|----------|
| 老课本（默认） | 装订边、网点纹理、衬线字体、油墨质感 | 教育类、内容类产品 |
| 手账本 | 虚线边框、马克笔色彩、手写字体、贴纸元素 | 生活类、笔记类产品 |
| 老报纸 | 复古排版、印刷网点、做旧纹理、黑体搭配 | 资讯类、文创类产品 |

### 3. 零配置自动化
- 自动识别现有页面结构，无需手动标注
- 自动生成适配的wxml/wxss代码，直接替换即可使用
- 自动保留原有业务逻辑，只改样式不影响功能
- 自动适配全局响应式，兼容不同屏幕尺寸

### 4. 质感交互优化
- 自动添加按压反馈：模拟纸张按压的下陷、变深效果
- 微动效：网点轻微浮动、装订边高亮等低存在感动效
- 印刷级排版：自动优化字重、行高、间距，符合阅读习惯

## 🚀 使用方法
### 基础用法
直接在对话中输入：
```
使用复古学院风美化 [页面路径]
```
示例：
```
使用复古学院风美化 pages/course/list
使用复古学院风美化 packageOrder/pages/confirm
```

### 自定义风格
指定风格变体：
```
使用手账风格美化 pages/profile/index
使用老报纸风格美化 pages/home/index
```

### 高级配置
可以自定义配置项：
```
美化pages/search/index，禁用网点纹理，装订边改为橙色，按钮用圆角样式
```

## 📋 配置项说明
| 配置项 | 可选值 | 默认值 | 说明 |
|--------|--------|--------|------|
| styleVariant | `textbook`/`notebook`/`newspaper` | `textbook` | 风格变体 |
| enableDotTexture | true/false | true | 是否启用网点背景纹理 |
| bindingEdgeColor | 颜色值 | `#2c3e50`（墨蓝）/`#c0392b`（砖红） | 左侧装订边颜色 |
| buttonStyle | `sharp`/`rounded` | `sharp` | 按钮样式：直角/小圆角 |
| enableSerifFont | true/false | true | 是否启用衬线字体（英文/数字） |
| animationLevel | `none`/`low`/`medium` | `low` | 动效等级：无/低/中 |

## 🎨 效果示例
### 美化前（AI常见风格）
- 大圆角卡片、渐变背景、玻璃态按钮
- 无排版层次，所有元素间距一致
- 悬浮动效浮夸，没有质感

### 美化后（复古学院风）
- 卡片左侧加墨蓝色装订边，小圆角，网点纹理背景
- 英文/数字用衬线字体，标题下划线，排版有层次
- 点击按压时卡片轻微下陷，装订边变砖红色，动效自然有质感
